PRINCIPALES ETIQUETAS HTML

ETIQUETAS DESCRIPCION
<body> Indica la parte del cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que indica donde empieza el contenido visible del documento.
<head> La parte superior del documento HTML, es donde podremos indicar los metadatos: título del documento, hojas de estilos, javaScript, CSS…
<div> Un elemento que es usado mayoritariamente para agrupar otros elementos y actuar como plantilla de otros controles. La etiqueta <div> nos ayuda a estructurar el documento en secciones.
<a> Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar. Otro atributo muy usado es target, el cual nos sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma.
<strong> Si tienes mucho texto, es importante poder dar énfasis a una parte en concreto, con la etiqueta strong lo podemos hacer.
<br> Con esta etiqueta HTML le podemos decir al navegador que viene un salto de línea. Nos sirve para hacer el texto más leíble.
<H1>..<H6> Hay diferentes niveles de títulos, del 1 al 6. Las etiquetas nos permiten indicar la importancia del título y para estructurar el contenido, de esta forma ayudamos a los bots a entender la importancia del contenido.
<ol>..<ul>..<li> Las etiquetas OL y LI nos sirven para crear listas, OL para listas ordenadas y UL para listas sin orden. Dentro de las listas, los elementos se identifican con la etiqueta LI.
<img> Usamos la etiqueta IMG para mostrar imágenes dentro del contenido. Necesita el atributo src para funcionar, ya que será donde indicaremos desde donde tiene que mostrar la imagen.
<p> Etiqueta que nos sirve para agrupar texto dentro de un parágrafo. El propósito es poder hacer el contenido más fácil de leer y organizado.

STYLES CSS DESCRIPCION SINTAXIS
<Colors> Establece el color del texto de un elemento. <color>
<Backgrounds> La propiedad background es un atajo para definir los valores individuales del fondo en una única regla CSS. Se puede usar background para definir los valores de una o de todas las propiedades siguientes: background-attachment,color, image, position, repeat. [<bg-layer<,] * <final-bg-layer>
<Borders> Propiedad abreviada para establecer el ancho, el estilo y el color del borde. <line-width< ||<estilo de línea> || <color>
<Margins> La propiedad CSS margin establece el margen para los cuatro lados. Es una abreviación para evitar tener que establecer cada lado por separado con las otras propiedades de margen: margin-top, margin-right, margin-bottom y margin-left. [<length< | <percentage> | auto ]{1,4}
<Text-align> Describe cómo el contenido en línea de un bloque se alinea horizontalmente si el contenido no llena completamente el cuadro de línea. start | end | left | right | center | justify | match-parent
<Fonts> Propiedad abreviada para configurar 'font-style', 'font-variant', 'font-weight', 'font-size', 'line-height' y 'font-family', en el mismo lugar de la hoja de estilo. La sintaxis de esta propiedad se basa en una notación taquigráfica tipográfica tradicional para establecer varias propiedades relacionadas con las fuentes. Syntax: [ [ <'font-style'> || <font-variant-css21> || <'font-weight'> || <'font-stretch'> ]? <'font-size'> [ / <'line-height'> ] ? <'font-family'> ] | caption | icon | menu | message-box | small-caption | status-bar
<Height> Especifica la altura del área de contenido, el área de relleno o el área del borde (según el 'tamaño de la caja') de ciertas cajas. <viewport-length>{1,2}
<Width> Especifica el ancho del área de contenido, área de relleno o área de borde (dependiendo del 'tamaño de la caja') de ciertas cajas. <viewport-length>{1,2}
<Links> Pseudo-clase que plica estilos a los enlaces que apuntan a páginas o recursos que aún no han sido visitados por el usuario, coincide con cada elemento no visitado. <a>, <area>, o <link> que tiene un atributo href.. :link
<Lists> La propiedad de estilo de lista (list-style) permite definir de golpe todos los parámetros: list-style-type, list-style-image, y list-style-position. list-style[ list-style-type || list-style-position || list-style-image ] | inherit
<Tables> Establece el algoritmo utilizado para diseñar celdas, filas y columnas. /* Keyword values */ table-layout: auto; table-layout: fixed; /* Global values */ table-layout: inherit; table-layout: initial; table-layout: unset;
<Position> La propiedad CSS de posición establece cómo se coloca un elemento en un documento. Las propiedades superior, derecha, inferior e izquierda determinan la ubicación final de los elementos posicionados. static | relative | absolute | sticky | fixed
<Overflow> La propiedad CSS overflow especifica: si recortar contenido, dibujar barras de desplazamiento o mostrar el contenido excedente en un elemento a nivel de bloque. [ visible | hidden | clip | scroll | auto ]{1,2}
< Float> Especifica cómo debe flotar una caja. Se puede configurar para cualquier elemento, pero solo se aplica a elementos que generan cajas que no están absolutamente posicionadas. left | right | none | inline-start | inline-end
